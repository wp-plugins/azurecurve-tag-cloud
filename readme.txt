Displays a tag cloud with easy control of settings and exclusion of tags from the cloud.

== Description ==
The plugin integrates with the standard WordPress tag cloud, but allows tags to be excluded and easy control of other settings such as min and max font sizes, the sort order and number of tags to include. 

This plugin is multisite compatible.

This plugin supports language translations. If you want to translate this plugin please sent the .po and .mo files to wordpress.translations@azurecurve.co.uk for inclusion in the next version (full credit will be given). The .pot fie is in the languages folder of the plugin and can also be downloaded from the plugin page on http://wordpress.azurecurve.co.uk.

== Installation ==
To install the plugin copy the <em>azurcurve-tag-cloud</em> folder into your plug-in directory and activate it. 

A Settings page allows all settings to be configured; deploy the widget to the sidebar of your choice to see the tag cloud.
